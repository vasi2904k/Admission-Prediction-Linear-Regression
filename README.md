# 🎓 Admission Prediction using Linear Regression

## 📌 Project Overview
This project aims to predict the probability of graduate admission based on academic and profile features such as GRE score, TOEFL score, CGPA, SOP rating, LOR rating, and University Rating.

This is a supervised machine learning regression problem since the target variable (Chance of Admit) is continuous.

---

## 📊 Dataset Information
The dataset contains student academic details and profile ratings.

### Features Used:
- GRE Score
- TOEFL Score
- University Rating
- SOP Rating
- LOR Rating
- CGPA
- Research Experience

### Target Variable:
- Chance of Admit

---

## 📈 Model Evaluation

The model was evaluated using:

- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- Root Mean Squared Error (RMSE)
- R² Score

### 📌 Model Performance:
R² Score: 0.780613892079095
RMSE: 0.07526809620765003

---

## 📊 Key Insights

- CGPA has the strongest impact on admission probability.
- Higher GRE and TOEFL scores positively influence admission chances.
- Academic performance plays a major role in predicting admissions.
